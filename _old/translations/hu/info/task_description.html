Nikoletta szeret mindent rendszerezni. Ezúttal egy számsorozatot rendszerezett és az erőfeszítései eredményeképpen egy mélyen egymásba ágyazott számokból álló listát kapott. Zsófia és István nem értik igazán az ő szervezési rendszerét és nekik arra lenne szükségük, hogy megértsék, hogy mindez mit jelent. Nekik szükségük van az Ön segítségére, hogy megértség Nikoletta őrült listáját.

Adott egy lista, amely egész számokat tartalmaz vagy egyéb beágyazott listákat, amelyek ugyancsak tartalmazhatnak más listákat, valamint egészeket, amelyek… egyszóval az alapelv érthető. Önnek az a feladata, hogy az összes egész értéket egy szimpla egyszintes listává alakítsa. A listaelemek sorrendjének változatlannak kell maradnia az eredeti lista balról jobbra olvasott karakterláncként megadott ábrázolásához képest.

Nekünk el kell rejteni ezt s programot Nikoletta elől úgy, hogy könnyen elrejthető kisméretűvé tesszük azt. Ennek okáért az Ön által megírt programkódnak 140 karakternél rövidebbnek kell lennie (a szóközökkel együtt).

Bemeneti adatok: egész számokból (int) álló egymásba ágyazott lista (list)

Kimeneti adatok: egy egydimenziós egész számokból (int) álló lista (list).

Példák:

flat_list([1, 2, 3]) == [1, 2, 3]
flat_list([1, [2, 2, 2], 4]) == [1, 2, 2, 2, 4]
flat_list([[[2]], [4, [5, 6, [6], 6, 6, 6], 7]]) == [2, 4, 5, 6, 6, 6, 6, 6, 7]
flat_list([-1, [1, [-2], 1], -1]) == [-1, 1, -2, 1, -1]

Alkalmazása: Az itt használt alapelv hasznos lehet bonyolult szerkezetű állományok feldolgozásánál és elemzésénél, valamint kreativitást igénylő programozási vetélkedő feladatainál, ahol kitétel a programkód rövidsége.
